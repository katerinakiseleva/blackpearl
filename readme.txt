== Description ==

Hi. I'm a starter theme called blackpearl, or underscores, if you like. I'm a theme meant for hacking so don't use me as a Parent Theme. Instead try turning me into the next, most awesome, WordPress theme out there. That's what I'm here for.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Instructions ==

Blackpearl Theme allows for two main customizations: Custom Options Page and Custom Post Type.

Custom options Page:
This customization provides you with the three ways to customize your website. The three ways are:

1. Website announcement. To add the announcement to the header of your website:
    - In your admin panel, go to Appearance -> Theme Options.
    - Now you will see the Option Name 'Site Announcements'.
    - In the text field following the Option Name, you can write you desired Website Announcement that will be displayed in the header.
    - Click 'Save Changes'.
2. Content area and Footer background color change. To change the color scheme:
    - In your admin panel, go to Appearance -> Theme Options.
    - Now you will see the Option Name 'Color selection for content'.
    - Click on the color you would like to choose: Pinkish Grey, Mint, Warm Beige, or Default Color.
    - Click 'Save Changes'.
3. Website title font change. To change the font:
    - In your admin panel, go to Appearance -> Theme Options.
    - Now you will see the Option Name 'Font selection for Site Title'.
    - Click on the font you would like to choose: Cinzel, Codystar, Petit Formal, or Default Font.
    - Click 'Save Changes'.


 Custom Query:
 This customization provides you with the ability to add a featured image from each of your posts onto the homepage.
    - In your Edit Post / Add New Post screen click on 'Set Featured Image' in the 'Featured image' tab.
    - Add the desired image to be displayed on the Home Page.
    - Check the 'images' category in the 'Categories' tab.
    - Click 'Save Changes'.
